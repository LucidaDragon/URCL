URCL_TryParseSource

Namespace: urcl.h

Declaration

bool URCL_TryParseSource(const URCL_Char* source, unsigned long length, URCL_InstructionInfo** instructions, URCL_LabelInfo** labels, URCL_InstructionInfo** headers, unsigned long* instructionCount, unsigned long* labelCount, unsigned long* headerCount)

Description

Attempts to parse the specified source string into instructions, labels, and headers. Returns true if the parsing finished successfully or false otherwise. Instruction, label, and header arrays are only valid if the result is true.
