URCL_ErrorInfo

Namespace: urcl.h

Declaration

struct URCL_ErrorInfo

Description

Represents an error that has occured in the parser.

Fields

HasError

True if an error has occured or false if an error has not occured.

Length

The length of the source region where the error occurred.

Message

A message describing the error.

Position

A pointer to the source position that the error occurred.
